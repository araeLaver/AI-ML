### 핵심 암기 노트

#### SageMaker 인스턴스 타입
```
인스턴스 타입 선택:
- ml.t3.*: 개발/테스트, 저비용
- ml.m5.*: 범용, 균형잡힌 성능
- ml.c5.*: 컴퓨팅 집약, 전처리/추론
- ml.p3/p4.*: GPU, 딥러닝 훈련
- ml.g4dn.*: 추론용 GPU, 비용효율적
- ml.inf1.*: AWS Inferentia, 추론 최적화
```

#### 데이터 형식
```
SageMaker 입력 형식:
- RecordIO-Protobuf: Built-in 알고리즘 최적
- CSV: 범용, 첫 열이 레이블 (헤더 없음)
- JSON/JSON Lines: 유연한 구조
- Parquet: 대용량 데이터, 컬럼 기반
- Pipe 모드: 대용량 데이터 스트리밍
```

#### 핵심 수치
```
기억할 수치들:
- SageMaker 기본 모델 크기 제한: 5GB
- Kinesis 레코드 최대 크기: 1MB
- Firehose 버퍼 크기: 1-128MB
- Lambda 타임아웃: 최대 15분
- Batch Transform 페이로드: 최대 100MB
- Serverless Inference: 최대 6GB 메모리
```
