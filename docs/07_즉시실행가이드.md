# 즉시 실행 가이드

## 오늘 당장 시작하기

### 1. API 키 발급 (30분)

#### OpenAI
1. https://platform.openai.com 접속
2. 회원가입/로그인
3. API Keys → Create new secret key
4. $5 충전 (초기 학습용 충분)

#### Anthropic Claude
1. https://console.anthropic.com 접속
2. 회원가입/로그인
3. API Keys → Create Key

### 2. 개발 환경 세팅 (1시간)

```bash
# Python 설치 확인
python --version  # 3.10+ 권장

# 프로젝트 폴더 생성
mkdir ai-ml-learning
cd ai-ml-learning

# 가상환경 생성
python -m venv venv

# 활성화 (Windows)
venv\Scripts\activate

# 활성화 (Mac/Linux)
source venv/bin/activate

# 기본 패키지 설치
pip install openai anthropic langchain langchain-openai python-dotenv jupyter
```

### 3. 환경 변수 설정

```bash
# .env 파일 생성
echo "OPENAI_API_KEY=sk-your-key-here" > .env
echo "ANTHROPIC_API_KEY=sk-ant-your-key-here" >> .env
```

### 4. 첫 번째 코드 실행

```python
# hello_llm.py
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()
client = OpenAI()

response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": "당신은 금융 전문가입니다."},
        {"role": "user", "content": "이상 거래 탐지의 핵심 원리를 설명해주세요."}
    ]
)

print(response.choices[0].message.content)
```

```bash
python hello_llm.py
```
